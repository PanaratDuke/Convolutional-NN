Project Description

In this project, Convolutional Neural Network (CNN) is an neural network which extracts or identifies a feature in a particular image. This forms one of the most fundamental operations in Machine Learning and is widely used as a base model in majority of Neural Networks like GoogleNet, VGG19 and others for various tasks such as Object Detection, Image Classification and others. CNN has the following five basic components: Convolution : to detect features in an image ReLU : to make the image smooth and make boundaries distinct Pooling : to help fix distored images Flattening : to turn the image into a suitable representation Full connection : to process the data in a neural network A CNN works in pretty much the same way an ANN works but since we are dealing with images, a CNN has more layers to it than an ANN. In an ANN, the input is a vector, however in a CNN, the input is a multi-channelled image.
